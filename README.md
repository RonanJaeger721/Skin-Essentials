# Skin Essentials

Premium static storefront for Skin Essentials, a skincare boutique on Nelson Mandela Avenue, Harare, Zimbabwe.

## Preview

Open `index.html` directly in a browser, or serve the folder locally:

```bash
python -m http.server 8173
```

Then visit `http://127.0.0.1:8173/`.

## Features

- Premium feminine storefront design using the Skin Essentials logo and favicon
- Real product visuals extracted from the provided catalogue screenshots
- Product search, category filters, cart drawer, and WhatsApp order link
- Interactive skin-routine builder
- Responsive desktop and mobile layout
